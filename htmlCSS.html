HTML & CSS

HTML и CSS являются основными языками для создания веб-страниц. HTML используется для структурирования и организации содержимого страницы, а CSS - для задания её визуального оформления. В данном обучении мы рассмотрим основы HTML и CSS, необходимые для создания веб-страниц курсов по веб-программированию.

HTML

HTML (HyperText Markup Language) - это язык разметки, используемый для создания структуры веб-страницы. Он состоит из набора тегов, которые определяют различные элементы страницы, такие как заголовки, параграфы, списки, ссылки, изображения и т.д.

Основы HTML

Ниже приведены основные элементы HTML, необходимые для создания веб-страниц:

  • !DOCTYPE html - это директива, которая определяет тип документа как HTML.
  • html - это корневой элемент, который содержит все другие элементы страницы.
  • head - это элемент, который содержит метаданные страницы, такие как заголовок, описание, ключевые слова, ссылки на стили и т.д.
  • title - это элемент, который задает заголовок страницы, который отображается в верхней части браузера.
  • body - это элемент, который содержит основное содержимое страницы, такое как текст, изображения, таблицы, формы и т.д.
  • h1-h6 - это элементы для создания заголовков разного уровня, от h1 (самый крупный) до h6 (самый мелкий).
  • p - это элемент для создания абзаца текста.
  • ul - это элемент для создания неупорядоченного списка (с маркерами) .
  • ol - это элемент для создания упорядоченного списка (с цифрами).
  • li - это элемент для создания элемента списка.
  • a - это элемент для создания ссылки. Он может содержать атрибут href, который задает адрес страницы или файла, на который ссылается ссылка.
  • img - это элемент для вставки изображения. Он может содержать атрибуты src (адрес изображения) и alt (альтернативный текст, который отображается, если изображение не может быть загружено).

Расширенные возможности HTML

HTML также поддерживает более продвинутые функции, такие как встроенные таблицы, формы, видео и аудио, а также элементы для создания интерактивных приложений, таких как canvas и svg. Однако, для этого уровня обучения мы остановимся на базовых элементах HTML.

CSS

CSS (Cascading Style Sheets) - это язык таблиц стилей, который используется для задания внешнего вида веб-страниц. Он позволяет определять стили для различных элементов страницы, таких как цвет, размер, шрифт, отступы и т.д. Стили можно задавать внутри тегов HTML, но на практике используют отдельный файл CSS.

Основы CSS

Пример CSS-кода

    Ниже приведены основные элементы CSS, необходимые для оформления веб-страниц:
  • Селекторы - это элементы CSS, которые определяют, какие элементы страницы будут оформлены. Например, h1 определяет стиль для всех элементов заголовка первого уровня (h1).
  • Свойства - это элементы CSS, которые задают стиль для выбранных элементов. Например, color задает цвет текста, а font-size задает размер шрифта.
  • Значения - это конкретные значения свойств, которые задаются для выбранных элементов. Например, color: red; задает красный цвет текста.

Как использовать HTML и CSS для создания веб-страниц

Чтобы создать веб-страницу, нужно создать файл HTML, в котором определены элементы, текст и изображения, которые будут на странице. Затем нужно создать файл CSS, в котором определены стили для каждого элемента на странице. HTML-файл может содержать как простой текст, так и специальные элементы, такие как заголовки, абзацы, списки и изображения. Каждый элемент на странице должен быть заключен в соответствующий HTML-тег. CSS-файл может содержать стили для каждого элемента на странице, определяющие цвет, размер, шрифт, отступы и другие параметры стиля. Стили могут быть определены для конкретного элемента, группы элементов или для всех элементов на странице. Когда HTML- и CSS-файлы готовы, нужно их связать между собой, чтобы стили из CSS-файла были применены к соответствующим элементам на HTML-странице. Для этого внутри тега head HTML-страницы нужно добавить ссылку на CSS-файл, используя элемент link.
